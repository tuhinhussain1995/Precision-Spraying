Project Summary

This project is dedicated to Precision Spraying, employing Deep Learning techniques such as YOLOv8s, alongside Flask and Angular frameworks. With 8000 images covering healthy crops, diseases, pests, and weeds, meticulous image labeling and processing are prioritized. Leveraging tools like labelimg for annotation, ultralytics, Tensorflow GPU for training, and OpenCV for processing ensures robust development. Additionally, flask_restful facilitates the creation of a RESTful web service for seamless integration. YOLOv8s, achieving an impressive 98.76% accuracy, outperforms other models like SSD MobileNet and YOLOv5. Efforts are focused on optimizing model training and system robustness, marking a significant stride in Precision Spraying technology and sustainable farming practices.
